🚀 QUICK START:

Local Build:
  chmod +x deploy.sh
  ./deploy.sh build
  ./deploy.sh up
  
CI/CD Setup (GitHub Actions):
  1. Add secrets: DOCKER_USERNAME, DOCKER_PASSWORD
  2. Add variable: LAST_BUILD_COMMIT (empty)
  3. Actions → "Build and Push Docker Image" → Run workflow
  
Use Pre-built Image:
  ./deploy.sh pull username/rl-swarm:latest
  ./deploy.sh up
  ./deploy.sh logs
